Installation
------------

.. code-block:: bash

   git clone https://github.com/pierre-24/advanced-python.git
   cd advanced-python
   # optional: use a virtualenv
   python -m venv venv
   source venv/bin/activate
   # install dev packages
   pip install -e.[dev]
